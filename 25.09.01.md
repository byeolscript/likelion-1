# CSS 인터렉션이 없는 것

- 고정된 스타일만 적용 됨  
- 사용자가 아무 행동을 해도 변하지 않음  

```html
<button class="static">버튼</button>

<style>
.static {
  background: gray;
  color: white;
  padding: 10px;
}
</style>

# CSS 인터렉션이 있는 것

- 사용자의 행동에 따라 스타일이 바뀜
- 대표적으로 :hover,:active,:focus 같은 가상(psedo-class)사용

```    <button class="interactive">버튼</button>

    <style>
      .interactive {
        background: gray;
        color: white;
        padding: 10px;
      }

      .interactive:hover {
        /* 마우스 올렸을 때 */
        background: blue;
      }

      .interactive:active {
        /* 클릭했을 때 */
        background: red;
      }

      .interactive:focus {
        /* 키보드로 선택됐을 때 */
        outline: 2px solid orange;
      }
    </style>
