# 25.09.01

## CSS 인터렉션이 없는 것

- 고정된 스타일만 적용 됨  
- 사용자가 아무 행동을 해도 변하지 않음  

```html
<button class="static">버튼</button>

<style>
.static {
  background: gray;
  color: white;
  padding: 10px;
}
</style>
```

## CSS 인터렉션이 있는 것

- 사용자의 행동에 따라 스타일이 바뀜
- 대표적으로 :hover,:active,:focus 같은 가상(psedo-class)사용

``` html
<button class="interactive">버튼</button>

    <style>
      .interactive {
        background: gray;
        color: white;
        padding: 10px;
      }

      .interactive:hover {
        /* 마우스 올렸을 때 */
        background: blue;
      }

      .interactive:active {
        /* 클릭했을 때 */
        background: red;
      }

      .interactive:focus {
        /* 키보드로 선택됐을 때 */
        outline: 2px solid orange;
      }
    </style>
```

## 아바타 컴포넌트 (온라인 상태)
```html
    <figure class="avatar-img">
      <img src="/src/assets/face/face1.jpg" alt="사용자 이름" />
      <figcaption><em class="sr-only">온라인</em><span class="on"></span></figcaption>
      <!-- figcaption 안의 상태 텍스트는 .sr-only로 숨겨 화면엔 안 보이고 보조기기엔 들림 
       <span class="on/off"> - 시각적으로 상태 표시 점 (초록=온라인, 회색=오프라인)-->
    </figure>
```

## 아바타 컴포넌트 (오프라인 상태)
``` html
   <figure class="avatar-img">
      <img src="/src/assets/face/face5.jpg" alt="사용자 이름" />
      <figcaption><em class="sr-only">오프라인</em><span class="off"></span></figcaption>
    </figure>
```
## 아바타 컴포넌트 (온라인,오프라인 상태) CSS
``` CSS
.avatar-img {
  background-color: #ffc; /* 영역 확인용 배경색 */
  --size: 64px; /* 아바타 크기를 변수로 관리 */

  width: var(--size);
  height: var(--size);
  position: relative; /* span 상태 점을 절대 위치로 배치하기 위함 */
}

/* 이미지 스타일 */
.avatar-img img {
  width: 100%;
  height: 100%;
  border-radius: 50%; /* 원형 이미지 */
}

/* 상태 점(span) 스타일 */
.avatar-img span {
  display: block;
  width: 28.125%;  /* 아바타 크기의 비율로 설정 */
  height: 28.125%;
  border-radius: 50%; /* 원형 */
  border: 2px solid #fff; /* 테두리 */
  position: absolute; /* 부모(.avatar-img) 기준으로 위치 지정 */
  right: 0;
  bottom: 0; /* 오른쪽 아래에 위치 */
}

/* 온라인/오프라인 색상 구분 */
.avatar-img .on {
  background-color: #4cfe88; /* 초록색 = 온라인 */
}

.avatar-img .off {
  background-color: #dbdbdb; /* 회색 = 오프라인 */
}
```
